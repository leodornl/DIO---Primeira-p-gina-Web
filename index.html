Ferramentas Utilizadas

A criação de páginas Web pode ser feita através de qualquer editor de textos, como por exemplo bloco de notas, notepad++ e etc., porém, foi orientado pelo professor a utilização do Visual Studio Code, uma IDE (Integrated Development Environment) da Microsoft opensource, que possui vários recursos interessantes. Orientou também, sobre a instalação de recursos adicionais (extensões) como Live Service e Emmet.


Inspetor de Elementos

Inspetor de Elementos é uma ferramenta disponibilizada pelo navegador Chrome (outros browsers também oferecem este recurso). O acesso a ela é simples, bastando pressionar com o navegador aberto F12 ou pelo caminho "mais ferramentas > ferramentas do desenvolvedor". Possui como principal característica, a inspeção de todo o conteúdo disponibilizado no navegador, contribuíndo desta forma, no entendimento sobre como foi realizada criação da página Web.

Estrutura Básica do HTML

Um arquivo .html possui como estrutura básica o delimitador principal html, que marca o início e final do documento, seguido pelo head, que é usado para delimitar o cabeçalho do documento e o body, que é usado para definir o corpo do documento, ou o que será mais visível para o usuário.

Falando Sobre Tags

No HTML, as tags possuem a função de marcar elementos, determinando desta forma, características específicas. As tags são delimitadas pelos símbolos <> , e o conteúdo dentro delas, dependerá do objetivo do desenvolvedor da página Web. Temos com exemplo as tags de cabeçalho, parágrafo, citação, marcação de texto, que permitem o uso de links internos e externos, para inserção de imagem, entre outras.

Atributos das Tags

Atributos são propriedades que uma tag pode ter, podendo alguns serem globais, e outros mais específicos. Por exemplo, na tag strong é possível acresentar o atributo id, que poderá ser utilizado pelo JavaScript para realizar alterações de forma mais otimizada. Já o atributo style (também dentro da tag strong), poderá ser usado pelo sistema CSS para uma melhor formatação visual do elemento selecionado. Como atributo mais específico, temos o type="text", utilizado na tag input, que irá criar um espaço na página Web destinado a formatação de texto (por exemplo: formulário de cadastro). Já o type="number", irá criar um campo numérico. Existe também o atributo type="color", que criará um campo para escolha e seleção de cores. Por fim, como atributo mais específico, temos o src=" " da tag img, que possibilita a adição de imagens (locais e hospedadas na internet).

Textos

Nesta sessão, foi ensinado as diferentes tags utilizadas na formatação e criação do texto que será lido e consumido pelo utilizador da página Web. Por exemplo, a tag h (1 ao 6) é responsável pela delimitação do cabeçalho e a numeração (1 ao 6), informa ao navegador a proporção em tamanho desejado. Para delimitar ou criar novos parágrafos, utilizamos a tag p. Já a tag utilizada para citação dentro do texto é a blockquote. Por fim, foi apresentado pelo professor a página Web w3schools (www.w3schools.com/html/html_formatting.asp), que traz uma excelente quantidade de informações relacionadas ao HTML e demais conteúdos ligados a área de desenvolvimento.

Listas Ordenadas e não Ordenadas

As listas são recursos muito utilizados para melhor organização do conteúdo. Foi apresentado pelo professor dois tipos principais, as ordenadas (através das tags ol li) e não ordenadas (através das tags ul li). A diferença consiste, basicamente, que as ordenadas apresentam algum tipo de ordenamento (principalmente numérico) enquanto que as não ordenadas, apresentam uma sequência gráfica (marcadores). É possível realizar a alteração dos marcadores dentro das listas não ordenadas via sistema CSS.

Links

Dentro do HTML, os links são utilizados para possibilitar a ligação de diferentes pontos ou áreas da página desenvolvida, pondendo direcionar a conteúdos locais ou a conteúdos externos da Internet. A tag utilizada é a href="". É possível também utilizar atributos que direcionam a abertura do link dentro da mesma aba já carregada (a href="" target="_self"), ou em uma nova aba do navegador (a href="" target="_blank"). Outro recurso interessante, é o atributo title (a href="" title=""), que abre uma caixa de sinalização quando o cursor do mouse passa por cima do link selecionado.
